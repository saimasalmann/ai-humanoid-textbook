# Tutorials

This section contains practical tutorials to help you get started with the textbook and implement the concepts covered in each module.

## Available Tutorials

- [Setup Guide](./setup-guide.md) - Complete setup instructions for the development environment
- [First Robot](./first-robot.md) - Building your first ROS 2 robot node
- [Debugging Tips](./debugging-tips.md) - Common issues and solutions when working with robotics systems

## Learning Objectives

After completing these tutorials, you will be able to:

1. Set up your development environment for robotics
2. Create and run your first robotic system
3. Debug common robotics development issues
4. Apply best practices for robotics development

## Exercises

1. Complete the setup guide and verify your development environment
2. Build and run your first simple robot node
3. Practice debugging techniques with the provided examples

## Summary

These tutorials provide hands-on experience with the fundamental concepts covered in the textbook. They guide you through setting up your environment, building your first robot, and troubleshooting common issues that arise in robotics development.

## Safety Disclaimer

When working with physical robotic systems during these tutorials, always follow proper safety protocols. Ensure proper supervision during all testing phases and maintain emergency stop capabilities.
